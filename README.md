# HR Employee Attrition SQL Analysis

## Descripción

Este proyecto consiste en el análisis de datos de **rotación de empleados (Employee Attrition)** utilizando **MySQL**.

Se construyó una base de datos relacional a partir del dataset de IBM HR Analytics para explorar los factores asociados a la permanencia y renuncia de los empleados mediante consultas SQL de nivel básico, intermedio y avanzado.

El objetivo fue aplicar SQL para responder preguntas de negocio relacionadas con Recursos Humanos y apoyar la toma de decisiones basada en datos.

---

# Objetivos

* Construir una base de datos en MySQL.
* Importar y organizar datos desde archivos CSV.
* Analizar los factores asociados a la rotación de empleados.
* Explorar diferencias entre departamentos, cargos y niveles organizacionales.
* Practicar consultas SQL utilizando funciones de agregación, agrupaciones y lógica condicional.

---

# Dataset

El conjunto de datos contiene información de **1.470 empleados** e incluye variables relacionadas con:

* Edad
* Departamento
* Cargo
* Nivel del puesto
* Salario mensual
* Horas extra
* Satisfacción laboral
* Equilibrio entre vida personal y trabajo
* Antigüedad en la empresa
* Distancia al trabajo
* Experiencia laboral
* Estado civil
* Educación
* Rotación del empleado

# Preguntas de negocio

Durante el proyecto se desarrollaron consultas para responder preguntas como:

* ¿Qué departamentos presentan mayor rotación?
* ¿Las horas extra aumentan la probabilidad de renuncia?
* ¿Qué cargos tienen los salarios promedio más altos?
* ¿La satisfacción laboral influye en la rotación?
* ¿Los empleados con mayor antigüedad permanecen más tiempo?
* ¿Qué nivel organizacional presenta el mayor ingreso promedio?
* ¿Existen diferencias salariales entre departamentos?
* ¿Qué características tienen los empleados que renuncian?

---

# Consultas desarrolladas

Entre las consultas realizadas destacan:

* Rotación por departamento.
* Rotación por cargo.
* Rotación por rango de edad.
* Rotación según horas extra.
* Rotación según satisfacción laboral.
* Salario promedio por departamento.
* Salario promedio por nivel del puesto.
* Antigüedad promedio según rotación.
